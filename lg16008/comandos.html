Comandos SQL esenciales

Los comandos SQL son las instrucciones que se usan para interactuar con una base de datos relacional. Se agrupan en diferentes categorías según la tarea que realizan. Nos enfocaremos en los comandos más comunes para gestionar los datos y la estructura de las tablas.

DML (Lenguaje de Manipulación de Datos)

Estos comandos permiten gestionar los datos dentro de las tablas.

SELECT

Función: Recupera datos de una o más tablas.

SELECT * FROM empleados;

Explicación: Muestra todas las columnas de la tabla empleados.

INSERT INTO

Función: Inserta nuevos registros (filas) en una tabla.

INSERT INTO empleados (nombre, apellido) VALUES ('Juan', 'Pérez');

Explicación: Añade un nuevo empleado con el nombre 'Juan' y el apellido 'Pérez'.

UPDATE

Función: Modifica los datos existentes en una tabla.

UPDATE empleados SET apellido = 'García' WHERE nombre = 'Juan';

Explicación: Cambia el apellido del empleado 'Juan' a 'García'.

DELETE

Función: Elimina registros de una tabla.

DELETE FROM empleados WHERE nombre = 'Juan';

Explicación: Borra de la tabla empleados el registro del empleado llamado 'Juan'.

DDL (Lenguaje de Definición de Datos)

Estos comandos se utilizan para definir la estructura de la base de datos y sus objetos.

CREATE TABLE

Función: Crea una nueva tabla en la base de datos.

CREATE TABLE productos (id INT, nombre VARCHAR(50), precio DECIMAL(10, 2));

Explicación: Crea una tabla llamada productos con tres columnas: id, nombre y precio.

ALTER TABLE

Función: Modifica la estructura de una tabla existente.

ALTER TABLE productos ADD COLUMN stock INT;

Explicación: Agrega una nueva columna llamada stock a la tabla productos.

DROP TABLE

Función: Elimina una tabla completa y todos los datos que contiene.

DROP TABLE productos;

Explicación: Elimina la tabla productos de la base de datos.
